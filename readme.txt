Allows you to use Tapsys Checkout with the WooCommerce plugin.

== Description ==

This is the official Tapsys Checkout plugin for WooCommerce. It allows you to accept credit cards and debit cards with the WooCommerce plugin. It uses a seamles integration, allowing the customer to pay on your website. This works across all browsers, and is compatible with the latest WooCommerce.


== Dependencies ==

1. Wordpress v3.9.2 and later
2. Woocommerce v3.1 and later
3. PHP v5.6.0 and later
4. php-curl extension

== Configuration ==

1. Visit the WooCommerce settings page, and click on the Checkout/Payment Gateways tab.
2. Click on Tapsys to edit the settings. If you do not see Tapsys in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
3. Enable the Payment Method, name it Pay with Credit & Debit Cards (this will show up on the payment page your customer sees), add in your API keys and Webhook Secrets for both Sandbox and Production environments.
4. Toggle between test payments and live payments by checking/unchecking the Enable Sandbox mode checkbox.

== Frequently Asked Questions ==

= Who can use this? =

Currently only merchants and store operators in Pakistan can use this plugin.

= Who can pay using this? =

Any customer with a valid Visa , MasterCard or Unionpay credit or debit card can make payments , We are adding more payment methods too . Stay Tuned !
